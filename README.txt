Задание: 
Реализовать консольное приложение "Заметки".

Каждая созданная заметка содержит:
а) идентификатор, 
б) заголовок,
в) тело заметки, 
г) дату/время создания или последнего изменения заметки.

Сохранение заметок необходимо сделать в формате json или csv формат
(разделение полей рекомендуется делать через точку с запятой).

Приложение должно уметь:
а) сохранять данные в файл,
б) читать данные из файла,
в) делать выборку по дате, 
г) выводить на экран выбранную запись, 
д) выводить на экран весь список записок,
е) добавлять записку,
ж) редактировать записку и 
з) удалять записку.
